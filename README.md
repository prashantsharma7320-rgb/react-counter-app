# React Counter App

A simple and responsive Counter App built using React.js.

## 🚀 Features
- Increase Counter
- Decrease Counter (Counter never goes below 0)
- Reset Counter
- Responsive UI

## 🛠 Tech Stack
- React.js
- JavaScript (ES6)
- CSS3
- Vite
